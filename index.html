Regions for iPhone

Location Based Reminders

Regions for iPhone is a simple app that reminds you to do things based on where you are. How simple is it?

  1. Create a new reminder
  2. Pick a location from the map
  3. That's it.

The next time you are roughly near the purple region, you will receive a notification reminding you of whatever it is that you need to do. You have the ability to specify how close you should be before receiving the reminder. Regions doesn't use GPS in the background, which means that you shouldn't notice any significant reduction in battery life.

Features

  • Location based reminders
  • Minimalist User Interface
  • Extremely low power consumption
  • Address Book Integration
  • Google Address geocoding
  • Specify the distance from your point of interest

Regions requires iPhone 4.
